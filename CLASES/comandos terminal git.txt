touch (nombre.tipo de archivo): crear un archivo en el directorio.

rm (nombre): borrar una carpeta/archivo.

cd (lugar donde quiero moverme): se utiliza para cambiar el directorio.

mkdir (nombre): crea un directorio.

python (nombre del programa): ejecuta el programa en la consola.

ls -a: nos deja ver carpetas ocultas en un directorio.
